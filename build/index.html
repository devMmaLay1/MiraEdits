🚀 MiraEdits Build Tool

Mobile-First Performance Optimization

📦 Production Build

Minify CSS and JavaScript files for optimal performance. This will create optimized versions in the dist/ folder.

📋 Build Instructions

For Node.js Build (Recommended):

1. Install dependencies: npm install
2. Run build: npm run build
3. Serve production: npm run serve:prod

Manual Optimization:

• Use online CSS minifiers (e.g., cssminifier.com)
• Use online JS minifiers (e.g., jscompress.com)
• Replace file references in index.html
